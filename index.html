Publications

RB-Modulation: Training-Free Personalization of Diffusion Models using Stochastic Optimal Control

NEW Litu Rout, Yujia Chen, Nathaniel Ruiz, Abhishek Kumar, Constantine Caramanis, Sanjay Shakkottai

We introduce Reference-Based Modulation (RB-Modulation), a training-free plug-and-play solution for content and style personalization. By incorporating style features into the controller's terminal cost, we modulate the drift field in diffusion models' reverse dynamics, enabling training-free personalization.

Beyond First-order Tweedie: Solving Inverse Problems using Latent Diffusion

NEW Litu Rout, Yujia Chen, Abhishek Kumar, Constantine Caramanis, Sanjay Shakkottai

We present an efficient second-order approximation using Tweedie’s formula to mitigate the bias incurred in widely used first-order samplers. With this method, we devise a surrogate loss function for every diffusion step to address inverse problems.
